1. Get data from log file link https://www.kaggle.com/datasets/tunguz/internet-firewall-data-set

   Fatih Ertam (2023) ได้วิเคราะห์ข้อมูลการบันทึกบนอุปกรณ์ไฟร์วอลล์และควบคุมการรับส่งข้อมูลทางอินเทอร์เน็ตตามผลการวิเคราะห์ที่มาจากการบันทึกการใช้อุปกรณ์ไฟร์วอลล์ของ Firat University ซึ่งจำแนกข้อมูลออกเป็น 4 คลาสโดยใช้วิธี SVM ได้แก่ Linear, Polynomial, Sigmoid และ RBF เพื่อให้ได้ผลลัพธ์ที่ดีที่สุดสำหรับการศึกษานี้ ประสิทธิภาพของตัวแยกประเภทได้รับการประเมินโดยใช้ความไว การเรียกคืน และ ค่าเฉลี่ยฮาร์มอนิกของคะแนน F1 เราสามารถระบุได้ว่าข้อมูลที่สร้างขึ้นจากการจัดหมวดหมู่มีความเกี่ยวข้องกับข้อมูลที่ตั้งใจไว้มากน้อยเพียงใด ในการศึกษานี้ ใช้ 11 ลักษณะ ในการวิเคราะห์เหตุการณ์ 65,532 ครั้ง ตามตาราง 
![Screenshot 2023-04-28 213450](https://user-images.githubusercontent.com/122291438/235176832-1482581c-8977-4479-a398-896acdf1ff95.png)
SVM ที่เลือกฟังก์ชันการเปิดใช้งาน Sigmoid 98.5% ให้ค่าการเรียกคืนสูงสุด ในขณะที่ตัวแยกประเภท SVM ที่มีฟังก์ชันการเปิดใช้งานเชิง Linear 67.5% ให้ค่าความแม่นยำสูงสุด ตามค่าคะแนน F1 พบว่าลักษณนามที่ใช้การเปิดใช้งาน 76.4% RBF นั้นให้ผลลัพธ์มากที่สุด เมื่อใช้ฟังก์ชันกระตุ้น Polynomial ในการจำแนกข้อมูล พบว่าทั้งค่าความแม่นยำและค่าการเรียกคืนอยู่ในระดับต่ำ เมื่อคำนึงถึงค่าเฉลี่ย จะเห็นได้ชัดว่าตัวแยกประเภทตามฟังก์ชันการเปิดใช้งาน RBF นั้นดีที่สุด สำหรับแต่ละคลาส เส้นโค้งลักษณะการทำงานของเครื่องรับ (ROC) ก็ถูกสร้างขึ้นเช่นกัน ดังนั้นจึงมีการนำเสนอการเปรียบเทียบด้วยภาพของอัตราบวกจริงและอัตราบวกปลอม การออกแบบระบบอัจฉริยะที่สามารถเลือกจัดการ
